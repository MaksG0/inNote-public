# inNote — Privacy Policy

**Effective Date: July 2, 2026**

### 1. General Provisions
This Privacy Policy describes how the **inNote** desktop application (hereinafter referred to as the "Application") processes user data. I deeply value your privacy and have designed the Application to ensure maximum data protection and security.

### 2. Data Collection and Processing
* **100% Local Storage:** The Application operates entirely offline and locally. All your sticky notes, text contents, and custom configurations are saved strictly on your local device in the user's local directory (`%AppData%`). Your data is never uploaded, transferred, or exposed to any external servers or third parties.
* **Zero Tracking and Analytics:** The Application does not contain any analytical systems, advertising trackers, telemetry, or hidden tools designed to collect personal information.
* **Internet Connection Usage:** The Application utilizes an internet connection solely for the built-in Update Checker feature. This process sends a quick, automated request to a public configuration file on GitHub to compare the current build number with the latest release. No personal data, hardware identifiers, or note contents are ever transmitted during this check.

### 3. Changes to This Policy
I reserve the right to update this Privacy Policy from time to time. The latest version will always be available on the official product platform pages.
